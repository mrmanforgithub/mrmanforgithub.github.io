前端基础学习网站搜集

因为想要亲自修改自己的博客，而不是完全照抄网上的教程，所以打算自学一下前端开发了。

本身我就是个软件工程专业的学生，之前学习过一节课叫做 html 网页开发，但是那只是浅尝辄止，基本上只知道了个大概，做出来的东西也比较的单薄。后续又用 servlet 加上 mysql，java 做后端，普通的 html+css+javascript 写了一个比较浅薄的书签管理网页，实在是太丑陋了所以这里不做展示。相比起来别人的课程作业多半使用 Vue，套网上的模板，做的都特别漂亮。我的作业完全就是原始人(捂脸)。不过那节课最后也是过了，毕竟是选修课(笑)。

这个博客也是之前突发奇想就照着教程一步步搭起来的，虽然完全就是做 copy 工作，但是不妨碍我对前端开发燃起了那么一丝的兴趣，所以我就打算自己学习一下前端。

学习前端其实只需要在 bilibili 上找那种时长二三十个小时的视频全部看完就行（直球）
只不过学习效率会很低,我本人是有普通的 html+css+js 的一些基础知识的
目前来说也就那些使用接口调用的 关于 promise 对象 关于 ajax 之类的异步同步操作的不是很懂

这里总结分享一下我学习前端的时候找到的几个有意思的网站，后续找到其他的就继续更新。

Mdn web docs

这个网站可以学习基础知识，了解基础元素的各项属性和特性，还有一些小题目让你做，可以检验你的学习成果。

HTML 的 https://developer.mozilla.org/zh-CN/docs/Web/HTML

CSS 的 https://developer.mozilla.org/zh-CN/docs/Web/CSS

JavaScript 的 https://developer.mozilla.org/zh-CN/docs/Web/JavaScript

ChocDocs

这个网站主要还是针对前端面试的，里面总结了一些面试题目和资源，但是好多教程都在知识星球上发布(想看？那就交 59 元门票)，这里面并没有，所以只是简单做个推荐，不算特别推荐

https://chodocs.cn/

RoadMap

这是个可以清楚告诉你应该学习什么的学习路线图，按照上面给的学习路线一步步学习你就能掌握大部分工作所需的内容了。

想当清晰明了，但是缺点是纯英文没有中文，看起来比较麻烦。

https://roadmap.sh/

CSS 小工具

可以提供你一系列的 css 格式，包括动画，字体等等

个别网站可能是国外网站，如果无法进入记得使用魔法

https://css.bqrdh.com/safety-color

https://navnav.co/

https://uiverse.io/

https://lingdaima.com/

手动调节的背景 UI 工具

https://neumorphism.io/#e0e0e0

前端代码平台

https://codepen.io/

颜色配色搭配

渐变色： https://uigradients.com/

中国传统颜色： https://www.zhongguose.com/#anyuzi

在线图片压缩工具

支持转换图片格式，压缩图片大小，不错的网站: https://imagestool.com/zh_CN/
